## 2. 서비스 핵심 컨셉

- **미래 예약형 감성 메시지 플랫폼**: 사용자가 특정 날짜에 공개되는 메시지(텍스트/영상)를 미리 남기는 서비스
- 핵심 감성: 감정 전달, 기대감, 시간이 만들어내는 연결
- 포지셔닝: 법률 서비스가 아닌 감정 전달 플랫폼 (타임캡슐 + 메시지 서비스)
- 주요 활용 사례:
  - 조부모 → 손자녀 유산 메시지
  - 부모 → 자녀 생일·입학·성인식 메시지
  - 커플 기념일 메시지
  - 군인 파병 전 가족에게 남기는 메시지
  - 임종을 앞둔 사람의 가족 메시지
  - 결혼 준비 중 파트너에게 남기는 메시지
  - 기업의 신제품 출시 이벤트 활용

---

## 3. 비즈니스 모델

### 무료 플랜
- 텍스트 메시지 및 외부 영상(유튜브 링크) 등록
- 기본 수신자 지정 및 링크 기반 페이지 생성
- 메시지 횟수 제한 (초과 시 유료 전환 유도 — GPT 모델 참고)
- 메시지 예약 기한 제한 (예: 1년 이내)

### 유료 플랜
- 플랫폼 내부 영상 저장
- 더 긴 예약 기한 (예: 5년, 10년)
- 강화된 개인정보 보호 및 전달 신뢰성
- 복수 수신자 및 백업 전달 옵션
- 메시지 존재 공개 여부 세밀 제어

### 플래티넘 / 기업 플랜
- 기업 이벤트 활용 (신제품 드롭, 직원 리텐션 인센티브 등)
- 팔로우 기능을 통한 기업 공개 메시지
- 파트너 업체 할인 혜택 (케이크, 꽃, 기프트카드 등 — 공급가 마진 활용)
- 패밀리 볼트 패키지 (가족 단위 공동 메시지함)
- 커플 패키지 (공동 기념일 알림 및 메시지)

## 4. 프리미엄 차별화 아이디어

- **실물 배송**: USB, 꽃다발, 케이크 등 파트너 업체와 연계한 실물 선물 옵션
- **기프트카드 예약 배송**: 파트너 기프트카드를 미래 날짜에 발송
- **기부 기능**: 사용자가 장학금, 식량 지원 등 카테고리를 선택해 기부 예약 (크라우드펀딩형)
- **특별 혜택 확장**: 중증 질환자 등 특정 상황에는 무료로 긴 기한 제공 → 바이럴 효과 기대

## 5. UX / 기능 아이디어

- **카운트다운 표시**: "D-30일", "D-1시간" 방식으로 개봉일까지 남은 시간 실시간 표시
- **개봉 알림**: 30일·1주일·1일 전 수신자에게 푸시/이메일 알림 발송
- **메시지 불변성**: 전송(저장) 확정 후 수정 불가 — 진심의 무게감 강조, 수정 이력 표시
- **카테고리 선택**: 가족, 연인, 친구, 자신에게, 기업, 공개 등 메시지 유형 분류
- **메시지 작성 가이드 (템플릿)**: 카테고리별 질문/주제 제시로 막막함 해소 (일기장 형식 참고)
  - 유료 회원에게 더 다양한 템플릿 제공, 월 구독자에게 신규 템플릿 정기 제공
- **썸네일**: 텍스트 메시지도 사진 또는 제목으로 썸네일 구성
- **공개/비공개 분리**: 메시지 존재는 공개, 내용은 개봉일까지 잠금
- **수신 거부 기능**: 악용 방지를 위한 수신 거부 및 계정 페널티 시스템
- **로그인**: 구글 등 서드파티 원클릭 로그인으로 진입 장벽 최소화

## 6. 기술 및 운영 고려사항

- **스토리지 비용**: 업로드/다운로드 과금 모델 활용 시 비용 절감 가능 (지정일에만 1회 접근)
- **데이터 보존 정책**: 기본 2~3년 보관 후 소프트 딜리트(실제 삭제는 1~2년 유예), 수신자가 내보내기 또는 유료 연장 선택 가능
- **보안 및 암호화**: 개인정보(이름·전화번호·주소) 암호화 필수, 콘텐츠 모니터링 방안 검토
- **신원 인증**: 초기에는 소셜 로그인으로 시작, 추후 외부 인증 기관 연동 고려
- **결제**: Stripe 등 전문 결제 시스템 사용, 건당 수수료 고려해 연간 구독 모델 유리
- **외부 링크(유튜브)**: 프라이버시 설정(링크 공유 전용)으로 무료 사용자 영상 처리
